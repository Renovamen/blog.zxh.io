介绍

Renovamen 2020-10-18

Gungnir 是一个基于 Jekyll 的博客主题。同时我现在正在尝试开发它的 VuePress 版本。

Github: Renovamen/jekyll-theme-gungnir

演示站点：jekyll-theme-gungnir.vercel.app

# 灵感来源

在 Huxpro/huxpro.github.io 的基础上瞎改的的主题，同时照搬借鉴了很多其他主题的代码和设计，包括但不限于 mashirozx/Sakura、kitian616/jekyll-TeXt-theme、Fechin/hexo-theme-diaspora、liuzc/LeaveIt 等（这么看来似乎也没多少东西是我自己写的了，逃）。

Gungnir，冈格尼尔，北欧神话中主神奥丁使用的用世界树树枝做成的武器，和本主题并没有什么关系但我就是用了这个名字。

# 特性

  • 响应式
  • 多级目录
  • 可切换的首页大图
  • 明亮和夜间模式：识别系统的外观设置并自动切换模式，也支持用户手动切换
  • 搜索（使用了 Simple-Jekyll-Search）
  • 文章目录（使用了 Tocbot）
  • 基于 Highlight.js 的代码高亮渲染，还支持代码块全屏和光标所在行高亮（从 WordPress 主题 Sakura 搬过来的功能）
  • 支持在文章中使用 Mathjax、Katex、Chart.js 和 mermaid
  • 文章中的一些附加样式（提示和标签）
  • 支持在文章中使用 emoji（使用了 jemoji）和其他表情（比如 B 站小电视）
  • 评论（Gitalk、Disqus 和 Valine）
  • 关于、作品和友链页

继承自原主题的特性：

  • PWA
  • RSS
  • 带标签的归档页
  • 文章多语言
  • 站点统计（Google Analytics 和 百度统计）
